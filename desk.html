--- layout: bare title: Desk reader view ---

A "bookmarklet" for Desk reader view. This will:

  • open all collapsed messages;
  • reverse the order so that it's chronological (oldest first).

This is much more convenient than the default view if you want to "read" a long ticket to follow a discussion. This feature will probably be added to Desk at some point in the future, but if you can't wait you can use it right now!

How to use it

Drag the below link to the bookmarks bar. If you don't have one add the contents of it to your bookmarks some other way I'm sure you're smart enough to figure out how).

Desk reader view

Warning

This is a hack. It will disable updating of tickets, and may have unpredictable side-effects if you try to modify stuff. You should probably modify stuff in a different tab or something and use this only for reading tickets.
